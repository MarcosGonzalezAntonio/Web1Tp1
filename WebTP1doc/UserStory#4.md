***Código:** #main-s1-us4*

***Nombre:** Construir página de login*

### **Detalle**

Como líder del Equipo de Diseño, quiero que la app disponga de una pantalla de *login* para permitir que las personas se identifiquen e inicien sesión. Esta pantalla será accesible desde /login.

![image.png](attachment:bcf8bd51-1048-450d-8277-ff2adc538551:image.png)

### **Escenarios**

- Dado que la persona usuaria debe identificarse para usar la tienda y aún no ha iniciado sesión, cuando llega a la página de login, debe poder ingresar su usuario y su contraseña en un formulario y luego hacer click en un botón con el texto ‘Iniciar Sesión’ y ser redirigida a la página de inicio.
- Dado que la persona usuaria debe identificarse para usar la tienda y aún no ha iniciado sesión, cuando llega a la página de login, y no posee credenciales debe poder dirigirse a la página /register.

### **Validación**

- La página tiene un input de texto para ingresar nombre de usuario.
- La página tiene un input de tipo contraseña para ingresar la contraseña.
- La página tiene un botón para iniciar el proceso de inicio de sesión.
- La página tiene un enlace para dirigir la navegación hacia la página de registro.
- La página tiene un enlace para dirigir la navegación hacia la página de inicio.
- Se hace uso correcto de las etiquetas HTML y las opciones de accesibilidad.
- Se respetan las buenas prácticas de usabilidad.
